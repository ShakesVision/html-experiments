# How to credit this project

If you use, fork, embed, or deploy tools from **html-experiments**, please attribute **Shakeeb Ahmad**.

## Required credit (pick one format)

**Short (footer / About page):**

> Based on tools by [Shakeeb](https://shakeeb.in)

**Long (README / documentation):**

> This project includes code from [html-experiments](https://shakeeb.in) by Shakeeb Ahmad, used under the Attribution Required License. See [LICENSE](LICENSE) and [NOTICE](NOTICE).

## Source redistribution

When sharing source code, include:

- `LICENSE`
- `NOTICE`
- This file (`ATTRIBUTION.md`) is optional but helpful

## In-app example (HTML)

```html
<footer>
  © Shakeeb ·
  <a href="https://shakeeb.in">shakeeb.in</a> ·
  <a href="../../LICENSE">License</a>
</footer>
```
