How to use

  1. Type your مصرع (line) and pick the bahr. The meter's adad digits become the beat grid — 2 = a long (held) syllable, 1 = a short one.
  2. Tap between letters to split the line into syllables; each chunk drops onto a beat-cell.
  3. Pick a Key (Sa) and a Raag/Scale, then Generate melody.
  4. Set the tempo (or tap it / use a mood preset) and the meend slider — left is abrupt note jumps, right glides smoothly between notes.
  5. Play to hear it as “laa” over a metronome. Edit any contour arrow, or ask for variations.

Contour arrows

  • → stay (same note)
  • ↑ / ↓ step up / down
  • ⇈ / ⇊ leap up / down
  • ∩ arch (up then settle)
  • ∪ dip (down then recover)

Terms

Beat / pulse — the steady tick. Meend — a smooth slide between notes (the qawwali “sine-wave” feel). Sargam — Sa Re Ga Ma Pa Dha Ni, the Indian solfège. Laghu/Guru (•/—) — short/long syllables that set the rhythm.
