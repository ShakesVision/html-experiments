What this does

A neutral way to map a live debate so the audience can actually follow the logic and see where each thread lands — instead of noise. Built for moderators and online creators.

1 · Set up

Name the topic and the two sides (colours identify who said what). Everything autosaves; reopen anytime.

2 · Add arguments

Type each argument, pick the side, language (English/Urdu), and reasoning type. Add a citation for naqli/authority points.

3 · Resolve each point

  • Support — opponent concedes
  • Counter — opponent rebuts (pick a type)
  • Split — break into sub-parts
  • Both — partly agreed, partly not

When a thread is settled, seal it: Resolved, Agree to disagree (the four-imams ikhtilaf), or No further discussion.

4 · Reasoning types

Deductive, inductive, abductive (best explanation), analogical (qiyas), causal, authority/naqli, empirical — and fallacy (mughalta) to flag bad moves (ad hominem, strawman, circular…).

5 · Present & document

Click a card to spotlight it for the audience. Present hides the console for a clean screen (Esc exits). Open threads keeps nothing forgotten over a long session; Export/Import and Print save the full transcript.
