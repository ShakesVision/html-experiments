## Development Tools

### Chrome DevTools

1. **Open DevTools**: `F12` or `Ctrl+Shift+I`
2. **Toggle Device Mode**: `Ctrl+Shift+M`
3. **Preset Devices**: 
   - iPhone SE (375px)
   - iPhone 13 (390px)
   - iPad (768px)
   - iPad Pro (1024px)
4. **Custom Size**: Enter any width you want
5. **Test Throttle**: Simulate slow networks

### Recommended Testing Sequence

```
1. Desktop (1180px) - baseline
2. Large tablet (1024px)
3. Tablet (768px)
4. Large phone (480px)
5. Standard phone (375px)
6. Small phone (320px)
```

## Responsive Image Tips

- Images scale with `max-width: 100%`
- Height auto-adjusts
- No distortion
- Proper aspect ratio maintained

## Common Issues & Fixes

| Issue | Cause | Fix |
|-------|-------|-----|
| Text too small | Font size not responsive | вң… Fixed with clamp() |
| Buttons overlap | Not enough space | вң… Added flex-wrap |
| Input won't zoom on iOS | Font < 16px | вң… All inputs 16px |
| Horizontal scroll | Width too wide | вң… Added calc(100vw - padding) |
| Toolbar wraps badly | Fixed gap | вң… Adjusted gap at breakpoints |
| Cards too big | Min-width not responsive | вң… Scale 150px вҶ’ 100px |
